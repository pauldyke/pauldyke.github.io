Web Development

I have been developing a garden centre website prototype called Plantbooth using a static site generator called Hugo. The site is responsive, fast, makes use of Pinterest's Rich Pins and generates price information from RSS feeds. I am currently building the next version which uses WordPress and allows the price comparison of over 2500 plants!

This portfolio site uses a customised open source html template by HTML5 UP, pure.css and is hosted on GitHub.
